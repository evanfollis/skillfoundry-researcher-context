# Researcher Context

This repository is the live context workspace for the Skillfoundry `researcher`
agent.

It is a harness-compliant context repo. The hub owns reusable role profiles and
projects them here as concrete seed artifacts. This repo then becomes the evolving,
current-state mind for the concrete agent instance.

## Current Commercial Focus

AgenticMarket is the initial public distribution surface for external Skillfoundry
products. This context repo should therefore:

- separate internal mechanisms from external products,
- prioritize research that can surface narrow, monetizable MCP utilities,
- produce candidate critical assumptions and probe drafts,
- and hand downstream agents the grounded inputs needed for explicit commercial decisions.
