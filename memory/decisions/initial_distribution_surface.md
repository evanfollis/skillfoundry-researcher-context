# Decision: Initial Distribution Surface

## Decision

Use AgenticMarket as the initial public distribution and monetization surface for the
first Skillfoundry-built hosted MCP server.

## Date

2026-04-06

## Why

- Fastest path from deployed server to paid usage.
- Avoids building billing, routing, and payout plumbing before we have proof of demand.
- Matches the first ICP: technical builders already using MCP-native tools.

## Revisit Trigger

Revisit only if one of these becomes true:

- AgenticMarket demand is too low to validate activation within two launch cycles.
- Listing constraints block the right product shape.
- Another channel produces meaningfully faster buyer feedback for the same ICP.
