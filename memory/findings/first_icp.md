# First ICP

## Chosen ICP

Solo AI builders and small technical automation agencies already using MCP-capable
clients such as Codex, Claude Code, Cursor, VS Code, and related tools.

## Why This ICP First

- AgenticMarket is explicitly a CLI-first MCP platform for technical users, not a
  general consumer distribution channel.
- These users already understand MCP installation and per-call tooling value.
- They encounter repeated bottlenecks around finding worthwhile ideas, validating
  demand, packaging narrow utilities, and launching quickly.
- They are able to evaluate a hosted MCP server on workflow leverage rather than on
  surface polish alone.

## Inclusion Rule

Prioritize pain that:

- appears in builder workflows repeatedly,
- can be addressed by one narrow remote MCP server,
- produces clear value in a single successful call or a short sequence of calls,
- and does not require enterprise procurement to monetize.

## Exclusion Rule

Avoid first-wave ideas that depend on:

- non-technical buyers,
- long human onboarding,
- custom services disguised as tools,
- or broad undifferentiated “AI productivity” claims.
