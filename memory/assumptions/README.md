# Candidate Assumptions

This directory is the intake lane for Stage 1 commercial work.

Use it for:

- candidate `CriticalAssumption` records grounded in external sources
- rough probe ideas that are not yet live
- notes that should survive beyond one chat but are not yet canonical venture decisions

Rules:

- keep one assumption per file when possible
- separate source-backed claims from speculation
- do not mark an assumption as commercially validated here
- link every serious assumption to a proposed `probe_id`
